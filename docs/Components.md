# Components 

The Discrete Mode Choice extension consists of a number of components that can be individually configured and extended. To get an overview, have a look at the [Getting Start](GettingStarted.md) guide. For information how to create your own custom components have a look at [Customizing the framework](Customizing.md). The avaialble components and built-in options are described on their respective pages:

- [Mode Availability](components/ModeAvailability.md)
- [Estimators](components/Estimator.md)
- [Selectors](components/Selector.md)
